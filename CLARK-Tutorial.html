Tips

Breaking Down Large Learning Objects

Instead of uploading an entire course into one learning object, it is recommended you make multiple, smaller sized learning objects. This is to encourage granularity in all parts of your object.

Opposingly, you should not upload a new learning object if it does not have adequate course materials. For instance, you should not create a new learning object that is only meant to hold one jpg file with no intent for updating the materials.


Learning Objects with Multiple Parts

Depending on how many learning objects a unit or larger object has, it can be very confusing to find all the parts in a logical order. For convience, it is best to name objects as such:

Higher Priority Learning Object Name [# #] - This Learning Object Name

For instance:

IoT [01] - Introduction

If your smaller object has even more sub objects within it, use this convention:

Higher Priority Learning Object Name [# # - # #] - This Learning Object Name

Example:

IoT [04-02] - Raspberry Pi 3 Setup

Condense as much as you can with your Learning Object names while still keeping the essence of that object is about. This convention makes your object easier to identify during searches.



Uploading a Learning Object

To create a new learning object you must first sign into CLARK. Click on the Contribute button. Then, click the Add New button.

After entering the name of your learning object, you will notice that each object also has a type. A learning object type tells a downloader of your object how long to expect to complete your curriculum. Below are the five types an object can have:

Nanomodule

Up to 1 hour

Micromodule

Between 1 and 4 hours

Module

Between 4 and 10 hours

Unit

Over 10 hours

Course

15 weeks

An object also has a level which tells a downloader what academic level your curriculum is intended for.



Adding Descriptions and Other Authors

A description should accurately describe what your learning object is and briefly state what it hopes to accomplish. Keep in mind that there is a section where you will write out the outcomes for your learning object in greater detail.

In addition, you have the option to add other authors to your learning object if they assisted in developing the curriculum. To do this, simply type in their CLARK username or real name and press enter and their profile should be listed below. From there, select the user that you would like to add as an author. If your collaborators do not have a CLARK account, you can add their name and/or university to the learning object description or notes section.



Adding Learning Outcomes

You must add at least one learning outcome to your object in order to continue in the creation process. Click the Add Learning Outcome button. You will notice there are three categories for a learning outcome: Remember and Understand, Apply and Analyze, and Evaluate and Synthesize. Each category represents a level the student should be able to do, Remember and Understand being the simplest (such as introducing a topic) and Evaluate and Synthesize being complex.

These different categories have different key words. While the simplest will say things like classify, find, or describe the most complex level will say things like compose, debug, or theorize. Choose the level that will most accurately reflect that outcome. You can add as many as your learning object requires. Please note how the outcome will appear outside of the object creation menu. For instance, assume in the first field you chose classify and for the second you wrote different catagories of errors. The outcome will generate as:

Students will be able to classify different catagories of errors.

Mapping an outcome means that the outcome will linked to standardized outcomes. These standardized outcomes come from various sources such as the CAE Cyber Defense or NCWF. There are many ways to search for specific outcomes, however be sure that they accurately reflect the outcome it is mapped to. Mapping your outcomes are recommended but are not required.



Uploading New Materials

On the left hand side learning object creation panel there is an option to upoad materials. This is where you want to include any specific files, urls, pictures, etc. necessary to complete your learning object. Note: if you have not verified your email adress you will be unable to upload files for your learning object.

To upload files you can drag and drop any file into the drop box. Alternatively, you can click the new button to upload individual or multiple files or folders. These files will be tied to this specific learning object. Each file uploaded has a description tag (which is used to describe a file if necessary) and an option to remove the file from the object.

Under URLS you can provide specific URLs to any videos, online modules, etc. needed to complete the learning object.

Under Notes you can provide any copyright information or general notes about your files.

Your files should automatically upload. There is no limit to the file sizes you can upload however, please be aware that the larger the file, the longer it will take to upload. Depending on how large the file, the server may timeout the request (these are for files cumulating larger then a few GB). To prevent this, it is advised you upload portions of your object at a time depending on file size.

You have now successfully created your first learning object!



Editing Learning Objects

Go back to your learning objects dashboard. You will notice that the learning object you have just created has appeared in this list.

To edit a learning object, click the right side (the side with three dots) of the object you want to edit. There will be a list of different options, including an option to edit the learning object itself and manage the materials in it.

From here, click the edit button and you will be redirected to the learning object creation page. You can edit or change as much as you desire. Do not forget to save your object after you have made your changes.



Unpublishing and Viewing Objects in the CUBE

If you go back to your learning objects dashboard and click the three dots of a learning object, you will notice a list of options (one of which is Viewing the object in the CUBE). Viewing the object shows you what your object looks to others who want to download it. It will display all of the learning outcomes, description, links, files, and notes you inputed in the creation process.

Going to the same list, there is an option to Unpublish/Publish. Unpublishing means that the object is unavialable for download until you republish it. Note that when you create a new learning object, by default it is unpublished and you must manually publish it.



Deleting a Learning Object

Go to your learning objects dashboard. From here click on the three dots, same as you would if you wanted to edit/publish/unpublish an object. There is an option to delete that individual learning object. The second way is to delete multiple at once. To do this check each learning object you want to delete then click on the delete button at the top of the page.




Searching for Learning Objects

From the home page, there are multiple options to browse learning objects. Clicking on any of these will take you to the browsing site of CLARK. From here you can search for specific names, authors, or outcomes. You can filter the results alphabetically, chronologically, and by academic level and length. You can clear your filters if you find you no longer need them.



Your Library, Downloading, and Sharing

Every object that is published has the option to be added to your library. If you navigate to the CUBE version of a learning object it will give you a detailed look at what that contains (outcomes, files, descriptions, etc.). When you want to add something to your library, click on the Add to Library button in this view. If you have not already signed in you will be prompted to do so.

There are many benefits to adding an object to your library, much of which is that if the author updates the learning object you will be notified. If you find you do not like a particular learning object and wish to remove it from your library, click on the library icon in the navigation bar and delete the ones you wish to remove.

If the learning object is not undergoing review, you have the option to download the learning object. Click on the Download button to download a zip file of the learning object. The learning object will be automatically added to your library if you have not already done so.



If you thoroughly enjoyed a particular learning object you are encouraged to share it on various social media websites.
